# CallawayPreOwnedService
## Purpose: 
### Windows Service that polls the Callaway PreOwned Service Product Search endpoint for wanted products and sends out e-mail alert when one of the target products is found.

## Stack:
### .NET Core 3.1
### SendGrid (E-Mail)
